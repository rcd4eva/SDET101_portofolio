Interactive Web Application for Data Analysis

The project uses Plotly Dash to create an interactive web application for analyzing data stored in a database directly from the browser. Users can easily filter the data through dropdown boxes with predefined parameters, dynamically updating the graphs as selections are made. Additionally, the application provides functionality to download the displayed graphs as images and export the filtered data as CSV files, enabling convenient reporting and further analysis.

MQTT to SQL Translator

A Python-based application that subscribes to an MQTT broker to store published data in a MariaDB database, and also monitors the database for new entries, publishing updated values back to relevant MQTT topics for subscribed devices.

Automated Data Collector

A Raspberry Pi Pico- ased device that collects various sensor data and publishes it to an MQTT broker via its built-in WiFi module. It features a HDMI touchscreen for real-time adjustment of data collection parameters. The device will use the MQTT broker as cloud-based EEPROM storage, restoring configuration settings automatically upon restart, and at the same time allowing for parameters from a web application connected to the MQTT broker. Device will use the local flash memory as caching to store data temporarily when the MQTT broker is unavailable.
